# Soyle-Qyzylorda
Soyle Qyzylorda — это полноценная веб-платформа, созданная для объединения местных событий и бизнесов в городе Кызылорда. Платформа поддерживает двухъязычный интерфейс (казахский/русский), персонализированные рекомендации на основе поведения пользователя, регистрацию на мероприятия и мгновенную публикацию заявок без модерации.
